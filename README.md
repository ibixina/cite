# Portfolio Website - Computational Cognitive Science

An impressive, interactive brutalist portfolio website themed around "Emergence of Intelligence" with advanced neural network visualizations.

## üé® Features

### Visual Effects
- **Enhanced Neural Network Background**: 80+ interactive nodes with:
  - Pulsing energy states
  - Dynamic signal propagation
  - Mouse interaction (nodes follow cursor)
  - Glowing connections
  - Real-time particle physics

### Interactive Elements
- **Custom Cursor**: Smooth-following cursor with interaction states
- **3D Card Tilts**: Project cards respond to mouse movement with 3D perspective
- **Glitch Text Effects**: Hover over title words for terminal-style glitch animation
- **Particle Explosions**: Hover spec boxes to trigger particle bursts
- **Smooth Scroll Animations**: Staggered fade-in animations as you scroll
- **Counter Animations**: Section numbers count up when scrolled into view
- **Scanline Effect**: Subtle green scanning line across the page
- **Grid Overlay**: Subtle grid pattern for depth

### Keyboard Shortcuts
- Press **N** to spawn a new neural node at cursor position
- Press **C** to reset the neural network

## üåê GitHub Pages Deployment

This site is ready for GitHub Pages:

1. Create a new repository (e.g., `portfolio`)
2. Push all files to the repository:
   ```bash
   git init
   git add index.html styles.css script.js
   git commit -m "Initial portfolio website"
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to repository Settings ‚Üí Pages
   - Source: Deploy from branch
   - Branch: `main` / `root`
   - Click Save

4. Your site will be live at: `https://YOUR_USERNAME.github.io/portfolio/`

## üé® Design Philosophy

**Brutalist Minimal Aesthetic**:
- High contrast black and white
- Neon green (#00ff00) accents
- Heavy 3px borders
- Bold typography (Archivo Black + Roboto Mono)
- Raw, computational feel

**Emergence Theme**:
The neural network background demonstrates emergence - complex patterns arising from simple rules. Each node follows basic physics, but together they create an organic, intelligent-looking system.
